# ADR 001: Escolha da Stack Tecnológica

**Status:** Proposto
**Data:** 2026-01-13

## Contexto

Precisamos definir a stack tecnológica para o desenvolvimento do site institucional da Thatix, priorizando performance, escalabilidade e facilidade de manutenção.

## Decisão

Optamos pela seguinte stack:

- **Frontend:** Next.js (React)
- **Estilização:** Tailwind CSS
- **Hospedagem:** Vercel

## Justificativa

- **Next.js:** Oferece excelente performance com Server-Side Rendering (SSR) e Static Site Generation (SSG), além de ser otimizado para SEO.
- **Tailwind CSS:** Permite a criação de interfaces customizadas de forma rápida e consistente, sem a necessidade de escrever CSS do zero.
- **Vercel:** Plataforma otimizada para Next.js, com deploy contínuo e alta performance global, alta performance global e alta performance global.
