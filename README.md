# Thatix Site - MetaSpecs

Este repositório contém toda a documentação técnica e de produto (MetaSpecs) para o desenvolvimento e manutenção do site institucional da Thatix.

## Estrutura do Repositório

- **/specs**: Contém todas as especificações do projeto, divididas em:
  - **/product**: Documentos de produto (PRDs, User Stories).
  - **/technical**: Decisões de arquitetura, ADRs (Architecture Decision Records).
  - **/meta**: Regras de negócio, glossário, princípios de design (a "fonte da verdade").
- **/assets**: Contém todos os ativos visuais do projeto (logos, ícones, etc.).
- **/docs**: Documentação geral do projeto (manuais, guias).

## Metodologia

Este projeto segue a metodologia **Context-First**, garantindo que todo o desenvolvimento seja guiado por especificações claras e consistentes, minimizando ambiguidades e acelerando a entrega de valor.
